Markdown a HTML

Convierte tus archivos Markdown a HTML en un solo clic con esta herramienta.

Arrastra y suelta un archivo o haz clic aquí para seleccionarlo

¿Qué es esto?

Esta aplicación, aparte de renderizar tus archivos Markdown en HTML, les aplica estilos elegantes y minimalistas, de modo que puedas imprimirlos, exportarlos a PDF con tu navegador o, simplemente, almacenarlos en tu ordenador.

Las últimas funcionalidades que he implementado han sido las siguientes:

  • El título del documento HTML será el mismo que hayas indicado como Cabecera 1 en tu archivo. En Markdown: # Title.
  • Añade el atributo target="_blank" a tus enlaces, excepto a aquellos que estén en el índice, siempre y cuando éste se encuentre al principio del documento.
  • Respeta las ToC o tablas de contenido que hayas creado en tu archivo Markdown. Siempre y cuando tus enlaces sean funcionales, lo serán también en el HTML generado. Incluso lo seguirán siendo si exportas el HTML a PDF. Por ejemplo, con Google Chrome.
  • Resalta la sintaxis de tus fragmentos de código SQL, JSON, Shell y Kotlin (por ahora). Basta con que especifiques el lenguaje tras indicar la apertura del código. Por ejemplo: ``` json.

Esta aplicación ha sido desarrollada usando únicamente HTML, CSS y JS. Además, utiliza las librerías marked.js y prism.js.
