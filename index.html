¿Cómo funciona?

Convierte y formatea

La aplicación permite cargar un archivo Markdown y convertirlo en un documento HTML con un formato elegante y minimalista. Utiliza la biblioteca "marked" para realizar la conversión y genera un HTML bien formateado a partir del contenido Markdown proporcionado.

Tabla de contenidos

La aplicación reconoce la estructura de encabezados (Table of Contents) en el archivo Markdown y crea un índice de navegación basado en ellos. Los enlaces en el índice son funcionales, lo que significa que al hacer clic en un enlace, el usuario se desplazará automáticamente a la sección correspondiente del documento HTML.

Enlaces externos

La aplicación identifica los enlaces que no forman parte del índice y los configura para abrirse en una nueva pestaña del navegador. Esto mejora la experiencia de usuario al permitir que los lectores exploren los enlaces externos sin perder el contexto del documento actual.

Resaltado de sintaxis

Si se detectan bloques de código en el archivo Markdown, la aplicación utiliza la biblioteca "Prism" para resaltar la sintaxis correspondiente al lenguaje de programación especificado. Esto mejora la legibilidad y comprensión del código al aplicar colores distintivos a las diferentes propiedades y elementos del lenguaje de programación utilizado.
